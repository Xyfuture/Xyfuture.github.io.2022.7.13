本次作业的主要内容是DFS和贪心算法,贪心算法看似简单但是证明存在一定难度

A 选数问题

ZJM友情客串

题意

题面是英文,但是比较容易理解,就是说给了一堆正数,让你从正数中选出几个数,使得他们相加得到给定的数

解法

解决方法就是DFS,感觉更像普通递归,递归函数 void dfs(int data[],int s,int l,int ans) 这个函数data是输入的数组,s是开始搜索的位置,s之前的都不能用,l是第几层,到达指定层之后就会停住,ans是上一层传来的答案.

这样的话,如果当前层到了指定层就从s开始扫描,如果+ans能等于给定值计数增加,扫描结束则退出,如果不是指定层,则从s开始扫描,进行递归dfs(data,i,l+1,ans+data[i])进入下一层

这样做的话能保证所有组合都是出现的,而且不会出现重复,为了优化性能,在到达指定层前,ans就超了指定答案时不进行递归.

code

#include <iostream>
#include <cstdio>
#include <cstring>

using namespace std;
int n,k,m;
int count = 0;//全局变量,总的次数
void dfs(int data[],int s,int l,int ans)
{
    if(l == k)//最后一个数,不用向下dfs了
    {
        for(int i=s;i<n;i++)
            if (ans+data[i] == m)
                count ++;   
        return ;
    }
    else
    {
        for(int i=s;i<n;i++)
            if(ans+data[i]<m)//省下来的数还能支撑继续dfs
                dfs(data,i+1,l+1,ans+data[i]);
        return ;
    }
    
}
int main()
{
    // int n,k,m;
    int c;
    int data[1000];
    memset(data,0,sizeof(data));
    scanf("%d",&c);
    for(int i=0;i<c;i++)
    {
        count = 0;
        scanf("%d%d%d",&n,&k,&m);
        for(int j=0;j<n;j++)
            scanf("%d",data+j);
        dfs(data,0,1,0);
        printf("%d\n",count);
    }
}

总结

降低时间复杂度的一个重要的方法是剪枝,可行性剪枝,最优化剪枝,同时加入vis数组避免重复访问.

B 区间选点

贪心

题意

给定了多个区间,区间之间是有可能重合的,求至少需要多少点才能使每一个区间内能有一个点

解法

这是一道贪心问题.首先我们进行排序,首先按照结束时间点,小的在前,如果结束时间点相同则按照开始时间点,小的在前.之后开始贪心,选取当前没有被标记的第一条线段,然后将点放在线段的结尾,更新标记情况,所有包含这个点的线段都被标记,然后重复循环,直到所有线段都被标记.最后输出标记的点数

证明

假设现在存在一组最优解

首先将所有的区间进行排序,然后按照点进行分组,一个点对应一组,组与组之间没有重复的区间,同时组与组之间满足规定的顺序. 对于一组而言,总有一个最小的区间(按照上面的排序方法),这个这个区间的末端一定是最小的,然后放置的点一定在末端及其之前,同时组中的其他区间一定能与这个区间重合,那么显然其他的区间的起始点在这个区间的末端前面,那么我们把点放在末端上必然能覆盖组中所有的区间 因此可以推证贪心能等效一个最优解因此贪心最优.

code

#include <cstdio>
#include <algorithm>
#include <cstring>

using namespace std;

struct block //记录线段
{
    int s;//起点
    int e;//终点
    int v;//标记
    bool operator < (const block & op)//重载<,用sort
    {   
        if(e<op.e)
            return true;
        else if(e == op.e)
            if(s>op.s)
                return true;
        return false;
        // return e<op.e;
    }
};

int main()
{
    block data[102];
    int n;
    memset(data,0,sizeof(data));
    scanf("%d",&n);
    for(int i=0;i<n;i++)
        scanf("%d%d",&data[i].s,&data[i].e);
    int count = 0;//被标记的线段数
    int ans = 0;//用的点数
    sort(data,data+n);//排序
    while(count!=n)
    {
        int cur = count;//因为是有序的,所以count之前的都被标记了
        for(int j=count;j<n;j++)
            if(data[j].s<=data[cur].e&&data[j].v!=1)//看看能去掉几个区间
                count++,data[j].v=1;
            else
                break;  //因为有序,可以用后面的覆盖,去了会TLE,但是我觉得去了时间复杂度也没有提升数量级
        ans++;
    }
    printf("%d\n",ans);
}

总结

贪心规则一开始我想的不对,我一开始想的是从重合部分最多的地方下手放点,但是这个很容易证明错误,但是居然过了OJ…后来怎么也证明不出来发现错了.

贪心法证明还是感觉有点难,有点玄学的感觉,一般用反正或者转化最优解的方法,实在证不出来还是先交几发看看猜的对不对吧,虽然过了OJ不说明做的一定对

C 区间覆盖

贪心

题意

这个题是让我们选区间覆盖给定的区间,找到选取区间最少的情况,注意一点区间是离散的,就是说(1,2) (3,4)是能覆盖(1,4)的,是离散的数字区间.

解法

解法还是贪心的原则,首先排序,按照区间起点从小到大排序,然后选出起点能与上一段接上的(起始时为0)的区间,然后从这些区间中选出终点最靠后的,然后用这个区间的终点为现在的长度,重复这个循环,如果能达到指定的长度则输出结果,如果长度不能被更新则输出-1

code

#include <cstdio>
#include <cstring>
#include <algorithm>
using namespace std;

pair<int,int> data[26000];
int visited[26000];

int main()
{
    int n,m;
    memset(data,0,sizeof(data));
    while(scanf("%d%d",&n,&m)!=EOF)
    {
        memset(visited,0,sizeof(visited));
        for(int i=0;i<n;i++)
            scanf("%d%d",&data[i].first,&data[i].second);
        int j = 0;
        int maxEnd=0;
        int count = 0;
        sort(data,data+n);
        while(j!=m)//达到条件退出
        {
            for(int t=0;t<n;t++ )
            {            
                if(data[t].first<=j+1)//满足当前区间能和总的区间连上
                {
                    if(visited[t]!=1)//没有被访问
                    {
                        visited[t]=1;//标记
                        if(data[t].second>maxEnd)//贪心最优
                            maxEnd = data[t].second;//更新
                    }
                }
                else
                    break;
            }
            count++;
            if(j<maxEnd)//更新下一步的起始点
                j = maxEnd;
            else//要是这次不能更新说明不能到达目标
                break;
        }
        if(maxEnd == m)
            printf("%d\n",count);
        else
            printf("-1\n");//无法覆盖
    }
}
